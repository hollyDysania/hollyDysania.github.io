生成SSH密钥添加到github

发表于 2018-07-17 | 分类于 工具
字数统计: 213 字 | 阅读时长 ≈ 1 分钟

一个新项目git push推送到GitHub远程仓库时偶尔会遇到需要输入账号密码登陆的情况。可以生成SSH密钥添加到GitHub，免密推送。

Step1检查本机现有的SSH密钥

如果有以下任意文件：

  • id_dsa.pub
  • id_ecdsa.pub
  • id_ed25519.pub
  • id_rsa.pub

直接跳至Step3

Step2创建一个新的SSH密钥

Step3复制SSH密钥

这段命令是复制到剪切板

也可以手动复制 路径为C:\Users\Administrator.ssh\id_rsa.pub

Step4添加到GitHub

GitHub

  • 在页面的用户栏的右上角，单击 Settings ；
  • 在左侧边栏点击 SSH and GPG keys.；
  • 点击 New SSH key ；
  • 写个title
  • 粘贴您的钥匙插入 Key 区域中；
  • 点击 Add SSH key ；
  • 确认通过输入操作GitHub的密码。
